# **App Name**: Discord Crew Application

## Core Features:

- Moderator Application Form: Gathers information from users interested in becoming moderators and sends responses to a specified email address. Each submitted application is stored for review.
- Administrator Application Form: Collects necessary data from candidates applying for admin roles and forwards it to a designated email address for evaluation.
- Email Submission Handling: Processes completed forms by sending them to a predefined email address.
- Form Moderation Interface: Provides authorized personnel with tools to view, and manage submitted applications.
- User Role Assignment: Allows admins to assign roles within the discord server upon reviewing and approving applications. This will require a discord bot and integration with the discord API

## Style Guidelines:

- Primary color: Deep violet (#673AB7), suggestive of authority and leadership
- Background color: Light violet (#F3E5F5), complementing the primary color to give a calm background
- Accent color: Teal (#009688), providing a contrast for calls to action
- Body and headline font: 'Inter', a sans-serif font known for its modern and clean appearance which pairs well with any user interface. 
- Code font: 'Source Code Pro' for any displayed code, monospaced to ensure readability.
- Use sharp, modern icons that reflect a tech-savvy environment, following Discord's existing style where appropriate.
- A clean and structured layout, optimized for readability and ease of navigation, will help guide users and admins smoothly.
